Instructions

Your mission, should you choose to accept it, is to build a simple Adapt module using the Content Team's instance of the authoring tool.

Above you'll find links to two 'script' pages (where each page represents an Adapt page) which you should use when building your module. You'll also notice a link to a complete version of the training module, which you can use for comparison.

1. Create your course

The first step is to head over to the authoring tool, and click Add new course. At this point, please include your name in your course name so that it can be easily identified.

2. Apply a theme and menu

You'll probably then want to add a menu and theme to your course. For this training exercise, please apply the standard Box Menu, and the Community theme

3. Get building

You now have everything you need to get started. A number of useful assets have been uploaded for your course building pleasure, so there should be no need to upload your own (although you can if you really want to).

Tips

  • Useful assets have been tagged training. You can use this to hide any other assets when using the asset manager.
  • Display titles are rendered in the content, whereas titles are only shown in the Page-level Progress.
  • Extensions must be added to a course in Manage extensions. Most extensions must also be specifically enabled for them to start working.
  • When on the Page Editor screen, you can use the arrows at the top of a component to switch its layout.

And that's all the help you're getting; the rest's up to you...

Extra Credit

So you've finished your module, and you want to dive a bit deeper, you keen bean. Here are some extra tasks for the brave:
  1. Set the dashboard image (hero image) for your course.
  2. Play around with the following block classes to add some visual interest to your pages:
    primary, secondary, tertiary, origami and transparent.
  3. Add Page-level Progress to your course.
  4. Add Resources to your course.
  5. Experiment with some of the other components which haven't been covered here.

Thanks for playing!

We welcome any (friendly) questions and feedback you might have.
