Title:

Text & Graphic

Body:

Adapt allows you to combine text and graphic components on the scrolling page to create rich and varied learning experiences. In addition, a wide range of interactive components are also available to help encourage deeper engagement with the material.

 

These presentation components can be structured in any way to help you create learning experiences that meet the needs of your learners.

Title:

Hot Graphic

Body:

You can add interactivity to an image by using the hot graphic. This component allows you to position icons over an image. When an icon is selected, content associated with its corresponding location is displayed in a window over the image. This component will fall back to a narrative when viewed on mobile.

This component is always spanned.

Title:

Accordion

Body:

You can use the accordion component to present learners with a series of headings, which once selected, expand to reveal associated text. Select each of the headings below to find out more about how accordions can be used.

 

These components can be either single or spanned. If spanned they can also contain a graphic.
